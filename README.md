\# Поздравлятор



\*\*Поздравлятор\*\* — SPA веб-приложение, сервер предоставляет АПИ (ASP.NET Core Web API), информация хранится в объектах, персистентность которых реализуется с помощью использования БД. Корневая страница выводит список сегодняшних и ближайших ДР, остальная функциональность доступна на отдельных страницах, ссылки на которые ведут с корневой. Необходимо реализовать хранение и отображение фотографий именинников.

\## Функциональность



Функциональность приложения - ведение списка дней рождения (далее ДР) друзей/знакомых/сотрудников, а именно: 

&#x20; • Отображение всего списка ДР (дополнительные возможности, такие как сортировка, выделение текущих и просроченных и т.п. - остаются на усмотрение реализующего)  

&#x20; • Отображение списка сегодняшних и ближайших ДР (дополнительные возможности, такие как сортировка, выделение текущих и просроченных и т.п. - остаются на усмотрение реализующего)  

&#x20; • Добавление записей в список ДР

&#x20; • Удаление записей из списка ДР

&#x20; • Редактирование записей в списке ДР 



\## Технологии



\### Backend

| Технология | Описание |

|------------|----------|
